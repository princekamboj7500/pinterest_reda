Pinterest Pin Configuration 

 

1 : Create Business Account on Pinterest 

2 : Create base64 of app_id:secret_key (you can find in Pinterest app manage) 

3 : Replace this base64 with PINTEREST_AUTH_KEY 

4 : Now Update your Redirect URIs on Pinterest account with Shopify preview link which is generated after “shopify app dev” command, You need to update it on every time you run this command, but once you go live on hosting you just put the hosted URL once. 

5 : Replace the PINTEREST_APP_ID with your. 

6 : Its important to change PINTEREST_API_HOST with https://api.pinterest.com/v5 for going live
